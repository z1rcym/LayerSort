# LayerSort

## インストール方法
`.jsxbin` ファイルを **ScriptUI Panels** フォルダに入れ、After Effects の「ウィンドウ」メニューから実行してください。

## 機能
選択しているレイヤーを、昇順または降順でナンバリングします。
